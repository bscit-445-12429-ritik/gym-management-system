How to use

  1. Upload a portrait using the Source Image panel on the left.
  2. Select the Document Type to auto-fill recommended size.
  3. Use Alignment controls to position and zoom the face inside the safe area.
  4. Tweak Enhance settings (brightness, contrast, B&W) as needed.
  5. Click Download Photo to save the cropped image or Save Print Sheet for multiples.

Tip: Choose Other in Document Type to enter a custom size.
